    kNN

k-Nearest Neighbors

    This is an example to see how kNN works.Here we assume a simple dataset
with 2 parameters to better visualize the data. (kNN can work on more
than 2 parameters but for understanding purpose we take only 2)The 2
parameters are population and number of deaths:*****Do not take this
example seriously*****A fatal disease has spread across many cities of
the world. The most dangerous thing about this disease is that it has no
symptoms and people just suddenly die.You are a super genius scientist
that found a cure to this disease, but there is no way to tell which
cities are actually affected by the disease.Your training set includes
1000 cities that you know are affected by the disease and the ones you
have cured, with parameters being: Population, Number of deaths in 1
month span before the cure was implemented.Now you have to find out of
the Test Set, which cities need the cure.

*****Training Set and Test Set randomly generated*****
